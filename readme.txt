Penjelasan Unguided 1

Program di atas membaca input suara sampai ditemukan angka 0, memvalidasi
suara yang hanya berada dalam rentang 1 hingga 20, lalu mencetak hasil 
jumlah suara masuk, suara sah, dan jumlah suara masing-masing kandidat yang 
mendapatkan suara.

Penjelasan Unguided 2

Program di atas kini menghitung Ketua RT dan Wakil Ketua RT 
berdasarkan suara terbanyak, dengan prioritas nomor kandidat 
terkecil jika jumlah suara sama.

Penjelasan Unguided 3

Program ini mencari apakah bilangan k ada dalam daftar, dan jika ada
, mencetak indeksnya. Jika tidak, mencetak "TIDAK ADA".
